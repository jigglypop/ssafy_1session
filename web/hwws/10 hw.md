4. 다음 예제를 통해 ‘후손 셀렉터’와 '자식 셀렉터’의 차이를 설명하시오.

   ```html
   <!-- 후손선택자. 모두 선택하는 선택자 -->
   div p {
   	color: crimson;
   }
   
   
   <!-- 자식선택자. 직계자식만 선택하는 선택자 -->
   div > p {
   	color: crimson;
   }
        
   ```
